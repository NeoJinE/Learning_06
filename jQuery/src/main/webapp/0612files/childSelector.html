자식/자손 선택자

  1. 자손 요소의 선택 방법
    • $(“선택자 선택자”)
    • $(“ol li”)
    • ol 의 자손인 모든 li 태그 선택 : 글자색(red)
  2. 자식 요소의 선택 방법
    • $(“선택자>선택자”)
    • $(“ol>li”)
    • ol의 자식인 li 태그 선택 : 글자색(blue)
    • $(“ul>li”)
    • ul의 자식인 li 태그 선택 : 글자(underline)
  3. 메소드를 사용하는 방법
    1. 자손 요소 선택 : find()
      • $(“선택자”).find(“선택자”)
      • $(“.method).find(“li”)
      • 클래스 method 모든 자손 li 태그 선택 : 글자색(black)
    2. 자식 요소 선택 : children()
      • $(“선택자”).children(“선택자”)
      • $(“.method).children(“li”)
      • 클래스 method 자식 선택 : 글자색(green)
